Indoor Localization using OptiTrack, PX4 Autopilot, and ROS 2

Project

Abstract

“How to use Motion Capture system like Optitrack with PX4 Autopilot and ROS 2 for indoor localization?” this was the main question of this hands-on workshop that was delivered over two days in the Karlovac university of applied sciences in Croatia. Two code repositories: mocap_px4_bridge offboard_control_example, and guideline document were published, as part of the training. This workshop and visit were part of the Horizon-Europe AeroSTREAM project.

Date
Oct 12, 2023 — Oct 13, 2023
Event
AeroSTREAM Workshop
Location
